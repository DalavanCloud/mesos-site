const std::string &  role = "*" 
) const

Finds a number of resources equal to toFind in these Resources and returns them marked with appropriate roles.

For each resource type, resources are first taken from the specified role, then from '*', then from any other role.

bool mesos::Resources::isAllocatable ( const Resource &  resource)
static

Returns true iff this resource is valid and allocatable.

In particular, a scalar is allocatable if it's value is greater than zero, a ranges is allocatable if there is at least one valid range in it, and a set is allocatable if it has at least one item.
