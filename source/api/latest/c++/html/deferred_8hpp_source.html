53 // The result of invoking the 'defer' routines is actually an internal
54 // type, effectively just a wrapper around the result of invoking
55 // 'std::tr1::bind'. However, we want the result of bind to be
56 // castable to a 'Deferred' but we don't want anyone to be able to
